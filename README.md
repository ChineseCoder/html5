# html5 & Vue Notes
```
<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8" />
		<title></title>
		<script src="js/vue.js" type="text/javascript" charset="utf-8"></script>
	</head>
	<body>
		<div id="app">
			{{message}}
		</div>
		<script type="text/javascript">
			let v=new Vue({
				el:'#app',
				data:{
					message:"Hello world !"
				}
			})
		</script>
	</body>
</html>
```
